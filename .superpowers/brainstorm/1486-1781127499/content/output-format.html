What format should the pitch deck output be in?

The tool extracts brand tokens — the output format determines how those tokens come alive.

BRAND PITCH

Self-contained HTML

A single .html file that opens in any browser. Uses the actual extracted colors and font stacks. Keyboard/click navigation between slides. Print-to-PDF anytime.

Pros

  • Pixel-perfect brand fidelity
  • No installs, one file
  • Print → PDF in one click
  • Loads web fonts directly

Cons

  • Not editable in Keynote/PPT
  • Requires browser to present
P
pitch-deck.pptx

PowerPoint / Keynote (PPTX)

A .pptx file generated via pptxgenjs. Opens in PowerPoint, Keynote, or Google Slides. Brand colors applied to shapes; system fonts only (no custom web fonts).

Pros

  • Editable by anyone
  • Native presentation apps
  • Easy to hand off

Cons

  • Can't load custom web fonts
  • Layout control is limited
  • Harder to match brand precisely
pitch.html
pitch.pptx

Both — HTML + PPTX

Generate both outputs from the same brand tokens. HTML is the fidelity reference; PPTX is the editable handoff. Adds complexity but covers all use cases.

Pros

  • Best of both worlds
  • HTML for presenting, PPTX to share

Cons

  • More to build and maintain
  • Two outputs to keep in sync

My recommendation: HTML — it's a personal tool, so brand fidelity matters more than editability. You can always print to PDF to share.
